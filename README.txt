
This folder contains 7 files, this README file, socialNetwork_simulator.py, and 5 text files for testing. 

socialNetwork_simulator.py creates a social networks using a 2D list. It can find common friends between users. It recommends friends to the user based on common friends. The simulator calculates the average number of friends in the network, the users with the most friends, and if a user knows everyone in the network. 

This project was created for an introductory course on Python development in November 2023.
